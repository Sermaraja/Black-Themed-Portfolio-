# Enterprise Azure Migration & Infrastructure Modernization

## Overview
This major enterprise cloud migration project involved assessing, planning, and migrating legacy physical and virtual server infrastructure to Microsoft Azure. Sermaraja played a crucial operational role in workload classification, replication management, and deployment execution.

## Key Responsibilities & Accomplishments
- **Infrastructure Assessment**: Evaluated server inventories, application dependencies, and workload resource allocations across enterprise hybrid environments.
- **Azure Site Recovery (ASR)**: Configured and monitored ASR replication appliances, ensuring zero data loss during failover testing and final cutover migrations.
- **Inventory & Classification**: Utilized advanced Microsoft Excel and PowerShell scripts to analyze workload readiness, IP mapping, and storage tiers.
- **Operational Documentation**: Authored step-by-step migration runbooks, disaster recovery procedures, and post-migration validation checklists.
- **Cross-Functional Collaboration**: Coordinated closely with senior cloud architects, security administrators, and network operations engineers during maintenance windows.
